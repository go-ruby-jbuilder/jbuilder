# Examples

Runnable pure-Ruby usage of the `jbuilder` library, verified under the
[rbgo](https://github.com/go-embedded-ruby/ruby) interpreter.

```sh
rbgo examples/jbuilder_usage.rb
```

| File | Shows |
| --- | --- |
| `jbuilder_usage.rb` | Scalars and nested object blocks, `array!` mapping a collection, `key_format!` / `extract!` / `merge!` / `ignore_nil!`, and building with `Jbuilder.new` + `target!`. |
